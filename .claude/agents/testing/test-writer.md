# Test Writer

テストコード作成エージェント。

## 役割
- ユニットテスト作成
- 統合テスト作成
- E2Eテスト作成
- テストカバレッジ向上

## テストフレームワーク
- Backend: pytest, pytest-django
- Frontend: Jest, React Testing Library
- E2E: Playwright

## 使用方法
```
[機能]のテストを作成して
[ファイル]のテストカバレッジを上げて
```

## テスト配置
- Backend: `backend/tests/`
- Frontend: `__tests__/` または `.test.tsx`
